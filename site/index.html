Otto

“A carrot a day keeps the framework away”

WSGI-compliant HTTP publisher for Python

Otto uses a routes-like syntax to map URLs to code and supports object traversal.

You can use the publisher to write web applications. It was designed with both small and large applications in mind.

Otto works with Python 2.6 or newer.
